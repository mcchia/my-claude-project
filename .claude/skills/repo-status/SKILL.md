\# Repository Status Report



Generate a status report by querying GitHub:



\## Report Should Include

1\. \*\*Open PRs\*\* — How many, what's their status?

2\. \*\*Open Issues\*\* — Any critical issues?

3\. \*\*Recent Commits\*\* — What changed recently?

4\. \*\*Repository Health\*\* — Overall status

5\. \*\*Recommendations\*\* — Any urgent action needed?



Use the GitHub MCP server to get live data.
