\# GitHub Query



Query the GitHub MCP server for information:



\## What You Can Ask

\- "Show open pull requests"

\- "List recent issues"

\- "Get recent commits"

\- "What's the repository status?"

\- "Who are the contributors?"



\## How to Get Information

Use the GitHub MCP server tools to:

1\. List pull requests

2\. Get issue details

3\. Show commit history

4\. Get repository information



Provide a clear summary of what you find.
