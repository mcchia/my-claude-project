\# Code Review



Review the file `$ARGUMENTS` and check for:



\## Review Checklist

\- Security problems (anything unsafe?)

\- Code quality (is it well-written?)

\- Best practices (does it follow standards?)

\- Clarity (is it easy to understand?)



\## How to Format Your Response



1\. \*\*Summary\*\* — One sentence about overall quality

2\. \*\*Issues Found\*\* — List any problems

3\. \*\*Recommendations\*\* — How to improve it

4\. \*\*What's Good\*\* — Positive things you found



Be helpful and constructive!
